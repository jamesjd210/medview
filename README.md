## MedView

### Setup
 - Clone this repository
 - `sudo apt install python3-virtualenv`
 - Change working directory to repo folder
 - `virtualenv .`
 - `source bin/activate`
 - `./installdeps`
 - Run server locally: `npm start` or `python3 main.py`
 - Run python tests: `pytest`
 - Build JavaScript: `npm run build`
